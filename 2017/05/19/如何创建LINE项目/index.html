如何创建LINE项目

发表于 2017-05-19 |
字数统计: | 阅读时长 ≈
  1. 首先，在LINE的github主页上下载源码（可以直接下载zip文件），并将内附linux文件夹下的东西解压到某个文件夹下，将该文件夹重命名为line。
    line
    • —-concatenate.cpp
    • —-line
    • —-line.cpp
    • —-normalize.cpp
    • —-reconstruct.cpp
    • —-train_youtube.sh
  2. 配置Linux环境下的编译环境
    1. Ubuntu默认安装了gcc。
    2. 下载gsl包文件，版本这里选择gsl-2.2.tar.gz，并解压。地址：ftp://ftp.gnu.org/gnu/gsl/
    3. 在解压后的gsl-2.2目录下执行以下操作
      • ./configure
      • make
      • make install
    4. Terminal下进入line目录,执行
      • g++ line.cpp -lpthread -lgsl -lgslcblas
    5. 将输出的a.out重命名为line。
    6. Terminal下执行./line，可能会报错
    7. 如果报错则在Terminal执行以下操作：
      • LD_LIBRARY_PATH=/usr/local/lib
      • export LD_LIBRARY_PATH
    8. 再次执行./line，如果提示参数信息则证明成功。
  3. 验证所给的net_youtube.txt 示例文件
    1. 下载youtube-links.txt.gz并解压。
      地址：http://socialnetworks.mpi-sws.mpg.de/data/youtube-links.txt.gz
    2. 重命名其中的release-youtube-links.txt为net_youtube.txt，复制一份至line目录
    3. 在line目录下执行：./train_youtube.sh
